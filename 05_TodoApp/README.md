# TodoApp

## Getting started
Install angular-cli using the following command
`npm install -g angular-cli`

Create a new app 
``` typescript
ng new todoApp 

ng serve
```

Create component
ng g component todo

Create service
 ng g service todo


Reference to angular CLI 
https://cli.angular.io/reference.pdf
